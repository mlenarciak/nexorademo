## Prerequisites

Before you begin, ensure you have:

<Steps>
  <Step title="Node.js 20+">
    Install Node.js from [nodejs.org](https://nodejs.org)
    
    ```bash
    node --version  # Should be v20.0.0 or higher
    ```
  </Step>
  
  <Step title="pnpm Package Manager">
    Install pnpm globally
    
    ```bash
    npm install -g pnpm
    ```
  </Step>
  
  <Step title="Neon Database">
    Create a free account at [neon.tech](https://neon.tech) and create a new database project
  </Step>
  
  <Step title="Clerk Account">
    Sign up at [clerk.com](https://clerk.com) for authentication
  </Step>
</Steps>

## Installation

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/mlenarciak/nexorademo.git nexora
    cd nexora
    ```
  </Step>
  
  <Step title="Install Dependencies">
    ```bash
    pnpm install
    ```
    
    This will install all dependencies across the monorepo.
  </Step>
  
  <Step title="Configure Environment Variables">
    Create `.env.local` in the project root:
    
    ```bash .env.local
    # Database
    DATABASE_URL="postgresql://user:pass@host.neon.tech/nexora?sslmode=require"
    DATABASE_URL_UNPOOLED="postgresql://user:pass@host.neon.tech/nexora?sslmode=require"
    
    # Clerk Authentication
    NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY="pk_test_..."
    CLERK_SECRET_KEY="sk_test_..."
    
    # App Configuration
    NEXT_PUBLIC_APP_URL="http://localhost:3000"
    NODE_ENV="development"
    ```
  </Step>
  
  <Step title="Setup Database">
    Generate Prisma Client and push schema to database:
    
    ```bash
    cd packages/database
    pnpm db:generate
    pnpm db:push
    pnpm db:seed
    ```
    
    This creates the database schema and populates it with test data.
  </Step>
  
  <Step title="Start Development Server">
    ```bash
    cd ../..
    pnpm dev
    ```
    
    Your app will be available at [http://localhost:3000](http://localhost:3000)
  </Step>
</Steps>

## Common Issues

<AccordionGroup>
  <Accordion title="Prisma Client Not Generating">
    ```bash
    cd packages/database
    pnpm db:generate
    ```
  </Accordion>
  
  <Accordion title="Migration Errors">
    Reset and try again:
    
    ```bash
    pnpm db:reset
    pnpm db:push
    pnpm db:seed
    ```
  </Accordion>
  
  <Accordion title="Clerk Authentication Issues">
    - Verify environment variables are set correctly
    - Check Clerk dashboard settings
    - Ensure URLs match between Clerk and your app
  </Accordion>
</AccordionGroup>

## Development Tools

### Prisma Studio
Browse and edit database records:
```bash
pnpm db:studio  # Opens http://localhost:5555
```

### Linting & Formatting
```bash
pnpm lint        # Check for linting errors
pnpm format      # Format code with Biome
pnpm type-check  # TypeScript validation
```

### Testing
```bash
pnpm test        # Run unit tests
pnpm test:e2e    # Run E2E tests
```

## What is Nexora?

Nexora is a modern, cloud-native hospitality property management system (PMS) designed for small to medium-sized properties in Italy and Brazil. Built on the Next-Forge framework with Neon PostgreSQL and Prisma ORM, Nexora provides a comprehensive solution for managing properties, rooms, bookings, payments, and guest services.

## Technology Stack

Nexora leverages the best modern web technologies:

- **Next.js 14** - React framework with App Router and Server Components
- **Neon DB** - Serverless PostgreSQL database
- **Prisma** - Type-safe ORM
- **Clerk** - Authentication and user management
- **shadcn/ui** - Beautiful, accessible UI components
- **Liveblocks** - Real-time collaboration
- **Metabase** - Embedded analytics

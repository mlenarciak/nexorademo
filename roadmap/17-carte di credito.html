Per impostare la prima PASSWORD per visualizzare le CARTE di CREDITO seguire i passaggi:

  1.  

  • IMPOSTAZIONI

  • GENERALI

  • cliccare su MODIFICA PASSWORD sulla riga corrispondente a PASSWORD CARTE DI CREDITO

  • PASSWORD ATTUALE → inserire la prima password di accesso, fornita alla creazione della licenza di Scidoo

  • NUOVA PASSWORD → inserire una nuova password che rispecchi i parametri forniti

  • CONTROLLA PASSWORD → reinserire la password creata in precedenza

  • cliccare su REIMPOSTA PASSWORD

  1.  

  • entrare in una prenotazione

  • cliccare in alto  CARTA con il pallino verde

  • cliccare sull'occhio (VISUALIZZA CARTA)

  • seguire i passaggi indicati

⚠️  la password delle carte di credito ha una validità di 3 mesi ed ogni volta che verrà modificata arriverà una mail con oggetto: FILE DI RECUPERO CARTE DI CREDITO; questo file è fondamentale per reimpostare la password in caso di smarrimento della stessa. Nel caso in cui non si dovesse trovare questo file e la struttura non dovesse ricordare la password, l'unico modo per reimpostarla è il RESET della password che comporterebbe la perdita di tutti i dati delle Carte di Credito presenti all'interno del gestionale.

 

⚠️ tutti i dati delle Carte di Credito rimarranno visibili all'interno della prenotazione fino a 3 giorni dopo il Check-Out del cliente; oltre a questa tempistica i dati verranno eliminati automaticamente e non saranno più recuperabili.

Per visualizzare i dati delle Carte di Credito, seguire i passaggi:

  1. entrare nella prenotazione

  2. cliccare sull'icona della Carta di Credito blu in alto (se saranno presenti i dati, apparirà un pallino verde)

  3. cliccare sull'occhio (VISUALIZZA CARTA)

  4. inserire la password

  5. CODICE DI AUTENTIFICAZIONE (se impostato, cliccare su RICHIEDI CODICE OTP, verrà inviata una mail alla struttura con il codice da inserire nell'apposita riga

  6. cliccare su VERIFICA

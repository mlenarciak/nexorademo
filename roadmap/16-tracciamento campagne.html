Istruzioni per il tracciamento dentro Scidoo

🎯 Obiettivo

Tracciare la provenienza di ogni richiesta ricevuta tramite il modulo contatti del sito dell’hotel, con dettagli su canali di accesso (es. Facebook Ads, Google Ads, campagne email, ecc.) e comportamento utente, utilizzando uno script sviluppato da Scidoo.

💻 Inserimento dello Script

Per attivare il tracciamento:

🔧 Script da inserire:

<script type="text/javascript" src="https://www.scidoo.com/preventivo_picker/widget_statistiche/include_widget.js" data-codice="**** INSERIRE QUI ID DELLA STRUTTURA***"></script>

 

📍 Dove va inserito:

All’interno del tag <head>
 

oppure

All’interno del tag <body>
 

➡️ Va inserito in tutte le pagine web interne del sito.

📂 Utilizzo avanzato con file PHP

Per un tracciamento ancora più accurato, è possibile integrare un file PHP nel sito:

Caricare il file PHP fornito da Scidoo nella directory principale del sito web.

Questo consente l’uso dei cookie per memorizzare e tracciare la provenienza delle richieste.

⚠️ Nota importante sulla privacy
In questo caso, è necessario aggiornare il modulo privacy del sito includendo una dicitura che autorizzi la condivisione dei cookie con Scidoo.

📈 Vantaggi del tracciamento

✅ La struttura sarà in grado di:

Rilevare la fonte precisa del traffico (es. Facebook, Google, newsletter, ecc.)

Conoscere la campagna specifica da cui proviene il cliente

Analizzare le conversioni generate da ogni canale di marketing

Creare report statistici affidabili per misurare il ROI delle campagne

📊 Metodo di tracciamento simile a quello di Google Analytics 4 (GA4), ma con tecnologia proprietaria Scidoo.

🔁 Esempio di utilizzo

Un cliente invia una richiesta dal sito.

Il sistema registra che è arrivato tramite campagna Facebook – Promo Estate 2025.

La struttura potrà attribuire quella conversione a quella campagna, ottimizzando gli investimenti futuri.

📬 Per assistenza tecnica o ricevere il file PHP:
Contattare il supporto Scidoo o il referente di progetto.

Check da fare per collegare una stampante termica alla rete

 

  1. Controllare che il jack dietro alla stampante termica lampeggi:

  2. Cercare un indirizzo IP libero 

  3. Download del programma che trovate in Generali → Dati Generali → (Se wifi) “scarica Setup wifi” (Se ethernet) “scarica”

  4. Eseguo come ammnistratore

  5. Inserisco: IP → SET , Wifi → nome (ssid), Wpa - Psk e Pw → SET

  6. Scidoo → Impostazioni generali → NUOVA STAMPANTE 

  • inserisco Ip →  🖉 → + aggiungo i prodotti 

  • parametri → taglio parziale

  • prodotti collegati → aggiungo i prodotti da collegare

  • i → PING

 

 

La stampante Custom KUBE termica ha di default il settaggio carta a 82,5mm.

Ciò porta alla stampa fuori carta.

Per settare la carta a 80mm aprire il browser e digitare nella barra indirizzi l'IP della stampante termica, poi andare nei settaggi e su Larghezza Carta ( in inglese paper width ) settare a 80mm e poi nel tasto in fondo cliccare sul tasto APPLY PRINTER SETTING per applicare i nuovi parametri.

 

 

Testare la stampa con il modulo ristorante stampando qualche riga di comanda

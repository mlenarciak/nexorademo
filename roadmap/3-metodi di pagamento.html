Per collegare VivaWallet abbiamo bisogno dei seguenti dati:

  • SOURCE CODE

  • MERCHANT ID

Per recuperare questi dati è possibile seguire i seguenti passaggi all'interno dell'account di Viva Wallet:

  • IMPOSTAZIONI 

  • ACCESSO API 

  • ID VENDITORE e copiare la stringa nella voce del metodo di pagamento dentro Scidoo in Mechant Id

  • VENDITE (o Saldi → Pagamenti Online)

  • SITI WEB/APP e creare il collegamento con Scidoo inserendo nel Dominio → www.scidoo.com → Protocollo HTTPS → se si vuole, si può sostituire il logo di Viva Wallet con quello della struttura

  • Inserire su “NOME DEL DOMINIO” → “www.scidoo.com” e selezionare “https”.

  • Incollare i seguenti url negli spazi corrispondenti: 

- E' ANDATO A BUON FINE → config/controllo_pagamenti/redirect_controllo_pagamento.php?service=21&success=1 

- NON E' ANDATO A BUON FINE → config/controllo_pagamenti/redirect_controllo_pagamento.php?service=21&success=0 

  • fleggare tutte le voci e accettare

  • Copiare il numero in alto a sinistra nella voce del metodo di pagamento dentro Scidoo in Source Code

 

Per poter attivare il prelievo da carta:

  • Dal metodo di pagamento dentro alle impostazione flaggare “Prelievo diretto”

  • Cambiare la password delle carte di credito da “impostazioni” → Generali → Modifica password

 

 

Per collegare PayPal abbiamo bisogno di questo dato:

  • CODICE CLIENTE

Per recuperarlo è possibile seguire i seguenti passaggi all'interno dell'account di PayPal:

  • SVILUPPATORE

  • LIVE

  • APP E CREDENZIALI 

  • creare L'App SCIDOO e copiare la stringa all'interno del codice cliente in Metodi di Pagamento

Per collegare Stripe abbiamo bisogno di questo dato:

  • CHIAVE PUBBLICABILE

  • CHIAVE SEGRETA

Per recuperarli è possibile seguire i seguenti passaggi all'interno dell'account di Stripe:

  • SVILUPPATORI (in basso a sinistra)

  • CHIAVI API e copiare la stringa dentro Scidoo in CHIAVE PUBBLICABILE

  • + CREA CHIAVE PRIVATA

  • CREARE LA TUA INTEGRAZIONE e far partire la mail alla struttura per l'autenticazione con successivo codice OTP

  • creare la chiave segreta inserendo il nome Scidoo e copiare la nuova stringa dentro Scidoo in CHIAVE SEGRETA

  • ALIAS

  • SECRET KEY (detta anche CHIAVE MAC)

Per collegare PayWay abbiamo bisogno dei seguenti dati:

  • TERMINAL_ID

  • SIGN_KEY

Per collegare Scala Pay abbiamo bisogno dei seguenti dati:

  • API KEY

Per collegare Worldline abbiamo bisogno dei seguenti dati:

  • Easy Checkout

  • Token

  • ApyKey

Per recuperare questi dati basta entrare su WorldLine cliccando sul seguente link: https://docs.axepta.it/ e seguire i passaggi:

  • CONFIGURE

  • PRODUCTION SOLUTIONS

  • copiare License Key Api su Api Key

  • copiare License Key Checkout su Easy Checkout Key

  • GENERATE ACCESS TOKEN → inserire le credenziali per generare il Token e copiarlo su Token

 

Per collegare Holipay oppure HeyLight abbiamo bisogno dei seguenti dati:

  • MERCHANT ID (Hotel ID)

  • SECRET KEY (Chiave privata holipay)

Per collegare Axerve abbiamo bisogno dei seguenti dati:

  • Api Key

  • Shop Login

Per recuperare i dati necessari al collegamento seguire i seguenti passaggi:

  • ENTRARE NELL’AREA RISERVATA e cliccare su APRE IMPOSTAZIONI (setting a sinistra) 

  • SICUREZZA

  • ACCESSI (ultima TAB) e creare le ApiKey

  • LIMITAZIONI IP mettere 217.61.36.74

  • SICUREZZA

  • LIMITAZIONI (attivare comunicazioni ApiKey e IP)
